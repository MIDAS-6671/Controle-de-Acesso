# Controle de Acesso ao Estoque - Projeto Wokwi Descrição --------- Projeto acadêmico que simula um sistema de controle de acesso ao estoque utilizando Arduino (sketch.ino), teclado matricial, display LCD e servo para tranca. Desenvolvido e testado no simulador Wokwi. Componentes ---------- - Arduino Uno - Teclado matricial 4x4 - Display LCD 16x2
